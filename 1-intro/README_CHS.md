# 介绍introduction

Furnace是能让你通过音源芯片（大多来自8bit或16bit时代）创作音乐（芯片音乐，chiptune）的工具。Furnace is a tool which allows you to create music using sound chips ("chiptune"), most from the 8/16-bit era.

它有一大堆特性和音源芯片。从NES，SNES到ES5506,VIC-20甚至街机，有这么多预设可选，Furnace大概率已经包含了你的目标芯片。it has a large selection of features and sound chips. from the NES, SNES and Genesis to ES5506, VIC-20 or even Arcade, Furnace has most likely covered your target with many presets to choose from.

每个芯片都用很多模拟器核心来模拟，因此Furnace产生的声音相对真实硬件挺逼真。every chip is emulated using many emulation cores, therefore the sound that Furnace produces is authentic to that of real hardware.

## 界面interface

Furnace使用一个音乐tracker的界面。就像一个写有音符的表格，然后表格向上滚动就播放了音符。Furnace uses a music tracker interface. think of a table with music notes written on it. then that table scrolls up and plays the notes.
在用furnace之前，看看[tracker概念与术语](concepts.md)以此了解tracker的界面。for an introduction to a tracker interface, see [tracker concepts and terms](concepts.md) before using Furnace.
也有一个[常见术语的词汇表](glossary.md)there's also a [glossary of common terms](glossary.md).

Furnace使用一个灵活的窗口系统，让你可以自由移动与组织。更多信息请见[2-界面](../2-interface/README.md)和[3-pattern](../3-pattern/README.md)Furnace uses a flexible windowing system which you may move around and organize. see [2-interface](../2-interface/README.md) and [3-pattern](../3-pattern/README.md) for more information.

once familiar with the tracker, look to [9-guides](../9-guides/README.md) for useful techniques.
