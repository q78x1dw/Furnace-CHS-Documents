# VERA 乐器编辑器instrument editor

VERA instrument editor consists of these macros:

- **音量Volume**: 音量序列volume sequence.
- **琶音Arpeggio**:音高序列 pitch sequence.
- **占空比Duty**: 方波的占空比序列pulse duty cycle sequence.
- **波形Waveform**:选择乐器使用哪个波形 select the waveform used by instrument.
- **声像Panning**: 切换左/右输出toggles left/right output.
- **音高Pitch**:精细音高 fine pitch.
