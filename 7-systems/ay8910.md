# 通用仪器General Instrument AY-3-8910

这个芯片被用于许多家用电脑(ZX Spectrum, MSX, Amstrad CPC, Atari ST, 等等),电子游戏机(Intellivision和Vectrex),街机基板甚至老虎机!this chip was used in many home computers (ZX Spectrum, MSX, Amstrad CPC, Atari ST, etc.), video game consoles (Intellivision and Vectrex), arcade boards and even slot machines!

这是个3通道的方波/噪声/包络声音发生器.这个芯片的强大的声音来自包络.it is a 3-channel square/noise/envelope sound generator. the chip's powerful sound comes from the envelope...

AY-3-8914变体在Intellivision中用到了,有每个通道的4级包络音量以及一个不同的寄存器格式.the AY-3-8914 variant was used in Intellivision, which is pretty much an AY with 4 level envelope volume per channel and different register format.

Furnace可以在AY-3-8910上面可以做软件采样播放,三个通道都可以播放4-bitPCM采样(代价是非常占用CPU)并且优化了CPU定时器效果,提供了各种调制(PWM和包络失真)效果,同样需要大量CPU资源.在导出为VGM之后利用定时器效果的乐曲,声音效果可能与想象中不同.Furnace is capable of doing software sample playback on AY-3-8910, where all 3 channels can play 4-bit PCM samples (at the cost of a very high CPU usage) and utilize CPU timer effects, providing all kinds of modulation (PWM and envelope distortion) effects, again at the cost of a higher CPU usage. Songs utilizing timer effects may not sound as intended after VGM export.

## effects

- `20xx`: **设置通道模式set channel mode.**
  - `0`: square方波
  - `1`: noise噪声
  - `2`: square and noise噪声和方波
  - `3`: envelope包络
  - `4`: envelope and square包络和方波
  - `5`: envelope and noise包络和噪声
  - `6`: envelope and square and noise包络,噪声和方波
  - `7`: nothing什么也没有
- `21xx`: **set noise frequency.** range is `0` to `1F`.
- `22xy`: **set envelope mode.设置包络模式**
  - `x` sets the envelope shape:设置包络
    - `0`: `\___` decay衰减
    - `4`: `/___` attack once起音后静音
    - `8`: `\\\\` saw锯齿波
    - `9`: `\___` decay衰减
    - `A`: `\/\/` inverse obelisco翻转的三角波('方尖碑')
    - `B`: `\¯¯¯` decay once衰减后音量保持最大
    - `C`: `////` inverse saw翻转的锯齿波
    - `D`: `/¯¯¯` attack起音
    - `E`: `/\/\` obelisco三角波('方尖碑')
    - `F`: `/___` attack once起音后静音
  - if `y` is 1 then the envelope will affect this channel.如果y是1那么包络会影响这个通道.
- `23xx`: **set envelope period low byte.设置包络周期低字节**
- `24xx`: **set envelope period high byte.设在包络周期高字节**
- `25xx`: **slide envelope period up.包络周期上滑**
- `26xx`: **slide envelope period down.包络周期下降**
- `29xy`: **enable auto-envelope mode.启用自动包络模式.**
  - in this mode the envelope period is set to the channel's notes, multiplied by a fraction.这个模式下包络周期会设置为本通道的音高乘上一个系数
  - `x` is the numerator.x是分子
  - `y` is the denominator.y是分母
  - if `x` or `y` are 0 this will disable auto-envelope mode.x或者y为0则停用自动包络模式
- `2Cxx`: **set timer period offset.设置定时器周期偏移**
  - bit 7 is the sign.bit 7是符号位
- `2Exx`: **write to I/O port A.向I/O端口A写入**
  - this changes the port's mode to "write". make sure you have connected something to it.这将端口的模式改为'写入'.确保你向那里连了东西.
- `2Fxx`: **write to I/O port B.向I/O端口B写入**
  - this changes the port's mode to "write". make sure you have connected something to it.这将端口的模式改为'写入'.确保你向那里连了东西.

## 什么是方尖碑what is obelisco

这是我对于一种从低处开始的尖锐波形的命名.it's a name I use for a spiky waveform that starts low.

他的源头是Deflemask之中一种恰好叫作方尖碑的波形.its origin is a wavetable that comes in DefleMask that happens to be called Obelisco.

## AY衍生品模式AY derivative modes

AY-3-8910太火爆了,被许多第三方仿制,有的有授权,有的没有.AY-3-810 was an absurdly popular chip that was blessed with many third-party clones, licensed or not.

- the AY-3-8914 variant was used in Intellivision, which is pretty much an 8910 with 4 level envelope volume per channel and different register format.
- 雅马哈YM2149是1983年发布的AY-3-8910复制品.它几乎与AYYamaha YM2149 was an AY-3-8910 clone released in 1983. it's almost identical to AY with minor differences being: higher hardware envelope step resolution (16 vs 32), half-clock mode when voltage level is low, much stronger DC offset and cleaner, but softer output.
- Sunsoft 5B是YM2149的翻版,半时钟模式强制开启.Sunsoft 5B is YM2149 clone with half-clock mode forced on.

## chip config

the following options are available in the Chip Manager window:

- **Clock rate**: sets the rate at which the chip will run.
- **Chip type**: changes the chip model.
  - AY-3-8910: the original model.
  - YM2149(F): Yamaha clone with higher envelope resolution and a different volume curve.
  - Sunsoft 5B: a special model found in one of Sunsoft's Famicom cartridges.
  - AY-3-8914: scrambled registers. has envelope volume control.
- **Stereo**: enable stereo output. channel 1 is output left, channel 2 is output center and channel 3 is output right.
  - **Separation**: allows you to set left/right separation.
- **Half Clock divider**: pulls the half clock pin, running the chip at half the speed.
  - only available in YM2149.
