# Yamaha YM2151

赋能多款街机基板,合成器,夏普X1/X68000的声音芯片.有8个4-opFM通道,有一个很nb的LFO和几乎没用过的噪音生成器.the sound chip powering several arcade boards, synthesizers and the Sharp X1/X68000. eight 4-op FM channels, with overpowered LFO and almost unused noise generator.

它也出现在几款pinball游戏机和当时的合成器,之后被著名的TX81Z使用的YM2414(OPZ)替代.it also was present on several pinball machines and synthesizers of the era, and later surpassed by the YM2414 (OPZ) present in the world-famous TX81Z.

在大多数使用他的街机基板中，这个芯片与一个PCM芯片结合使用，例如[SegaPCM](segapcm.md) 或[OKI的ADPCM产品线](msm6295.md).in most arcade boards the chip was used in combination with a PCM chip, like [SegaPCM](segapcm.md) or [OKI's line of ADPCM chips](msm6295.md).

## 效果effects

- `10xx`: **设置通道8运算器4的噪音频率set noise frequency of channel 8 operator 4.** `00`禁用噪音,`01`到`20`启用噪音.  `00` disables noise while `01` to `20` enables it.
- `11xx`: **设置通道的反馈set feedback of channel.**
- `12xx`:**设置运算器1的TLset operator 1 level.**
- `13xx`: **设置运算器2的TLset operator 2 level.**
- `14xx`: **设置运算器3的TLset operator 3 level.**
- `15xx`: **设置运算器4的TLset operator 4 level.**
- `16xy`:设置运算器的倍频 **set multiplier of operator.**
  - `x` 是运算器is the operator (1-4).
  - `y` 是新的MULT值is the new MULT value..
- `17xx`: **设置LFO速度set LFO speed.**
- `18xx`: **设置LFO波形set LFO waveform.**
  - `00`: 锯齿波saw
  - `01`: 方波square
  - `02`: 三角波triangle
  - `03`: 噪音noise
- `19xx`: **设置所有运算器的attack rate set attack of all operators.**
- `1Axx`: **设置运算器1的attack rate set attack of operator 1.**
- `1Bxx`: **设置运算器2的attack rate set attack of operator 2.**
- `1Cxx`: **设置运算器3的attack rate set attack of operator 3.**
- `1Dxx`: **设置运算器4的attack rateset attack of operator 4.**
- `1Exx`: **设置LFO AM深度set LFO AM depth.**
- `1Fxx`: **设置LFO PM深度set LFO PM depth.**
- `30xx`: **启用包络硬件重置enable envelope hard reset.**
  - 这通过在新音符之前插入一个快速释放和短的延时来实现this works by inserting a quick release and tiny delay before a new note.
- `50xy`: **设置运算器的AM set AM of operator.**
  - `x`是运算器,值`0`意味着"所有运算器"`x` is the operator (1-4). a value of 0 means "all operators".
  - `y`决定AM是否开启`y` determines whether AM is on.
- `51xy`: **设置运算器的SLset SL of operator.**
  - `x`是运算器,值`0`意味着"所有运算器"`x` is the operator (1-4). a value of 0 means "all operators".
  - `y`是值`y` is the value.
- `52xy`: **设置运算器的RRset RR of operator.**
  - `x` is the operator (1-4). a value of 0 means "all operators".同上
  - `y` is the value.
- `53xy`: **设置运算器的DTset DT of operator.**
  - `x` is the operator (1-4). a value of 0 means "all operators".同上
  - `y` is the value:
    - `0`: +0
    - `1`: +1
    - `2`: +2
    - `3`: +3
    - `4`: -0
    - `5`: -3
    - `6`: -2
    - `7`: -1
- `54xy`: **设置运算器的RS set RS of operator.**
  - `x` is the operator (1-4). a value of 0 means "all operators".同上
  - `y` is the value.
- `55xy`: **设置运算器的DT2 set DT2 of operator.**
  - `x` is the operator (1-4). a value of 0 means "all operators".同上
  - `y` is the value.
- `56xx`: **设置所有运算器的DRset DR of all operators.**
- `57xx`: **设置运算器1的DR set DR of operator 1.**
- `58xx`: **设置运算器2的DR set DR of operator 2.**
- `59xx`: **设置运算器3的DR set DR of operator 3.**
- `5Axx`: **设置运算器4的DR set DR of operator 4.**
- `5Bxx`: **设置所有运算器的D2Rset D2R/SR of all operators.**
- `5Cxx`: ****设置运算器1的D2Rset D2R/SR of operator 1.**
- `5Dxx`: ****设置运算器2的DRset D2R/SR of operator 2.**
- `5Exx`: ****设置运算器3的DRset D2R/SR of operator 3.**
- `5Fxx`: ****设置运算器4的DRset D2R/SR of operator 4.**
- `60xy`: **设置运算器掩码(?就是各个运算器的开关)set operator mask.**
  - 打开/关闭运算器enables or disables operators.
  - 如果x是0,y取值0~f,他就是一个位掩码,y是以下活跃运算器的位的和.if `x` is `0`, `y` ranges from `0` to `F`. it is a bitfield, so `y` is the sum of the active operators' bits:
    - OP1 is +1, OP2 is +2, OP3 is +4, and OP4 is +8.
    - 例如,只打开OP2和OP4就是2+8=10,导致`xy`的值是`0A`for example, having only OP2 and OP4 on would be 2 + 8 = 10, resulting in an `xy` value of `0A`.
  - 如果x是1到4,这个效果只针对于那个运算器;y为`0`把它关闭,`y`为1把它打开.if `x` is `1` to `4`, the effect targets that operator; `y` turns it off with a value of `0` and on with a value of `1`.
    - for example, the effect `6031` enables OP3.
- `61xx`: **设置算法set algorithm** (0 to 7).
- `62xx`: **设置LFO FM深度set LFO FM depth** (0 to 7).
- `63xx`: **设置LFO AM深度set LFO AM depth** (0 to 3).
